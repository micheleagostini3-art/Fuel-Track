# ⛽ Fuel Tracker

App mobile per tracciare spese carburante e service. **100% offline**.

## ✨ Funzionalità

- 📱 Interfaccia mobile-first
- 🔌 Funzionamento offline
- 💾 Salvataggio automatico
- ⛽ Benzina, Diesel, Metano
- 🔧 Service e manutenzione
- 📊 Statistiche in tempo reale
- 📥 Esportazione TXT

## 📥 Installazione

1. Scarica l'APK da [Releases](https://github.com/micheleagostini3-artciao/fuel-expense-tracker/releases)
2. Installa sul tuo telefono Android
3. Apri e inizia a registrare!

## 🎯 Come usare

1. Seleziona data
2. Scegli tipo (Benzina/Diesel/Metano/Service)
3. Compila i dati
4. Clicca Salva
5. Il file TXT si aggiorna automaticamente

## 💾 I dati vengono salvati in:

- LocalStorage (app)
- File TXT (Documents/spese-auto.txt)
